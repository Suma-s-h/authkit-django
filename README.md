# 🔐 AuthKit Django

A modern authentication and user management system built with Django. The project includes secure authentication, profile management, password updates, email verification status, and a responsive user dashboard.

---

## ✨ Features

- User Registration
- Secure Login & Logout
- User Dashboard
- Profile Management
- Edit Profile
- Change Password
- Email Verification Status
- Django Admin Panel
- Responsive UI
- Authentication System

---

## 🛠 Tech Stack

- Python
- Django
- HTML5
- CSS3
- Bootstrap
- SQLite

## 🚀 Installation

```bash
git clone https://github.com/Suma-s-h/authkit-django.git

cd authkit-django

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

Open:

```
http://localhost:8000/
```
